Presentamos el atlas

ATLAS

El robot humanoide más dinámico. El sistema de control avanzado de Atlas y el hardware de última generación le dan al robot el poder y el equilibrio para demostrar una agilidad a nivel humano.

ALTO VOLTAJE
Atlas tiene uno de los sistemas hidráulicos móviles más compactos del mundo. Una batería personalizada, válvulas y una unidad de potencia hidráulica compacta permiten que Atlas entregue alta potencia a cualquiera de sus 28 juntas hidráulicas para impresionantes proezas de movilidad.
DINÁMICA
El sistema de control avanzado de Atlas permite una locomoción muy diversa y ágil, mientras que los algoritmos razonan a través de interacciones dinámicas complejas que involucran todo el cuerpo y el entorno para planificar los movimientos.La cámara térmica integrada opcional le permite detectar puntos calientes que indican problemas en tanques, maquinaria o conductores eléctricos.
LIGERO
Atlas utiliza piezas impresas en 3D para darle la relación fuerza-peso necesaria para saltos y volteretas.
